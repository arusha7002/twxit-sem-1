# twxit-sem-1

commands in makefile: 
execute command : 'make clean' to rebuild all files 
execute command : 'make' to compile 
execute command : './twixt' to run the program

input format is : row column 
valid row and column entries are specified in the print statements as you run the program 
error messages are displayed incase invalid moves are played 
printing of message 'GAME OVER' indicates end of game 
player 1 is RED denoted by a red 'R' in the grid player 2 is BLACK denoted by a grey 'B' in the grid
